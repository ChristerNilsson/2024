# Fördelar

* Man vet alltid var man ska sitta.
* Man behöver inte känna den man ska spela med. 
* Färgväxling sker automatiskt. Vänd aldrig på något bräde!
* Man rör sig alltid medurs. Låt hjärtat visa vägen!

# Blanketter
